# 🌸 Mongle UI

**몽글몽글한 감성의 따뜻한 React 컴포넌트 라이브러리**

---

## Philosophy

- 🌸 **Fluffy** — Soft and cozy, rounded corners, gentle shadows
- ❤️ **Warm** — Warm tones that feel welcoming
- 🎨 **Emotional** — Expressive colors and smooth animations
- 🏠 **Cozy** — Components that feel like home

---

## Design Tokens

**Colors**

- Primary: Coral `#FF8B6D`
- Secondary: Periwinkle `#818CF8`
- Success: Mint `#14B8A6`

**Typography**

- Primary font: LINE Seed Sans KR
- Fallback: -apple-system, sans-serif

---

## Tech Stack

- React 19 + TypeScript (strict)
- TailwindCSS
- Storybook 8
- Vitest 2 + Vite 6

---

## Component Categories

- **General** — Button, Typography, Icon
- **Form** — Input, Select, Checkbox, Textarea
- **Layout** — Card, Container, Stack, Grid, Divider
- **Feedback** — Alert, Toast, Badge, Spinner
- **Overlay** — Modal, Drawer, Tooltip, Popover
- **Navigation** — Tabs, Breadcrumb, Link
- **Utilities** — Portal, Transition, VisuallyHidden, FocusTrap

---

## Design Principles

- Tailwind only — no custom CSS files, no inline styles
- Semantic props — `disabled` not `isDisabled`, `className` not `sx`
- ARIA-first — all interactive components include accessibility attributes
- WCAG 2.1 AA — minimum 4.5:1 color contrast ratio
