# Project Overview

## Identity

**Name:** Mongle UI  
**Type:** React Component Library  
**Concept:** 몽글몽글한 감성의 따뜻한 컴포넌트

## Philosophy

- **🌸 Fluffy:** 부드럽고 포근한
- **❤️ Warm:** 따뜻한
- **🎨 Emotional:** 감성적인
- **🏠 Cozy:** 아늑한

## Colors

- Primary: Coral (#FF8B6D)
- Secondary: Periwinkle (#818CF8)
- Success: Mint (#14B8A6)

## Font

- Primary: LINE Seed Sans KR (둥근 느낌, 몽글몽글 감성)
- Fallback: system fonts (-apple-system, sans-serif)

## Tech Stack

- TypeScript
- React 19
- TailwindCSS
- Storybook 8
- Vitest 2
- Vite 6

## Rules

### ✅ Always

- Tailwind only
- TypeScript strict
- React.memo
- ARIA attributes
- Design tokens

### ❌ Never

- Custom CSS files
- Inline styles
- Hardcoded colors
- `any` type
